pycheron.rollseis
=================

This package includes are functions that are utilized within some of the above functions but may be useful on their own
for correcting data issues if decided to go that route (rolling/filter functions) as they are rolling statistical
functions. All of these codes have been adapted from the seismicRoll package. [#]_
